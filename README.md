# 🚀 Task Manager API

REST API для управления задачами с автотестами.

---

## 📌 О проекте

Небольшое backend-приложение на FastAPI с SQLite базой данных.  
Реализован полный CRUD для задач и покрыт автотестами с использованием pytest.

---

## 🛠 Стек

- Python
- FastAPI
- SQLite
- SQLAlchemy
- pytest
- httpx

---

## ✨ Функциональность

- Создание задачи
- Получение списка задач
- Получение задачи по ID
- Обновление задачи
- Удаление задачи
- Валидация входных данных
- Автотесты

## ▶️ Запуск приложения

Установить зависимости:

    pip install -r requirements.txt

Запустить сервер:

    uvicorn app.main:app --reload

После запуска открыть:

👉 http://127.0.0.1:8000/docs

---

## 🧪 Запуск тестов

    pytest -q
